Installer Windows 7

Sur les les ordinateurs les plus anciens et mêmes les plus récents

Installer Windows 7 c'est très simple

Si vous voulez installer windows 7 sur un pc plus récents c'est juste en dessous.

Publié le 18 décembre 2021

Pour les pc plus vieux (BIOS legacy)

Pour installer Windows 7 sur Bios legacy c'est le plus facile, il faut télécharger depuis ce lien https://massgrave.dev/windows_7_links. Je recommande d'installer la version Ultimate(Édition Intégrale). Il vous faudra ensuite Ventoy et une clé usb de 4GB minimum. Puis télécharger la version ventoy-1.1.05-windows.zip ou similaire, vérifier bien que l'extension soit en .zip ! Puis aller dans votre explorateur de fichier, faites clique droit sur le fichier télécharger et appuyer sur extraire. Ouvrez le dossier jusqu'a trouver le fichier Ventoy2Disk, double cliquer dessus, puis cliquer sur option -> Partition Configuration et sélectionnez NTFS, puis sur install. Copier le fichier fr_windows_7_ultimate_with_sp1_x64_dvd_u_677299 à à votre clé usb qui doit s'appeller Ventoy, brancher la clé usb à l'ordinateur ou vous voulez installer windows. Et maintenant vous êtes prêt à Installer Windows 7 !

Pour les pc plus récents (UEFI)

Pour installer Windows 7 sur les pc plus récent c'est plus compliqué mais c'est faisable. Pour ceci il vous faudra Ventoy, Ntlite, 7-zip, l'iso de Windows 7 (Ultimate en x64), KB2864202,Les pilotes USB 3.x, KB2990214, KB2990941,KB3087873 et KB2990214 pour le support Nvme. Installer Ventoy sur votre clé usb (4GB minimum) en utilisant le NTFS pour partition configuration (dans options), Puis installer les logiciels Ntlite et 7-zip,

Il vous faudra ensuite télécharger l'iso de windows 7 en x64
